# Artofficial

A full MERN stack [AI image generator](https://www.artofficial.fun/) powered by Cloudinary and DALL-E-3; built with React, MongoDB, Express, Node, and Hostinger.

![banner_image](/client/src/assets/artofficial.jpg)

## Technologies Used
* OpenAI's DALL-E-3
* Cloudinary
* ReactJS
* MongoDB
* Express
* NodeJS
* Hostinger
